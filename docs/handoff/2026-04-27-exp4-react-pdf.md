# Handoff — 2026-04-27 — Experiment 4 React-PDF Lab Report

## Why this exists

The first iteration of the Exp 4 lab-report export produced a formal-but-plain
Matplotlib `PdfPages` PDF. The user requested that the formatting match
**Experiment 1's** Lab_Report_Angular_Momentum.pdf — i.e. the CUHK-Shenzhen
academic style with Times-Roman fonts, LaTeX-rendered equations, top/mid/bottom-
rule academic tables, a cover page, and a running header. This handoff records
the second iteration that uses the **same toolchain as Exp 1** to render the
PDF directly in the browser.

## What changed

### New file

- `frontend/src/components/Exp4ReportPDF.tsx` (new, ~480 LOC)
  - Uses `@react-pdf/renderer` (already a dependency, used by Exp 1 / 2 / 3 / 6 / 7).
  - LaTeX equations rendered via `https://latex.codecogs.com/png.image?...`.
  - Times-Roman / Times-Bold / Times-Italic only — no sans-serif.
  - Reusable `T` table component with the academic 1.5-pt top, 0.6-pt mid, and
    1.5-pt bottom rules used by Exp 1.
  - Reusable `Fig` component with caption + image fallback.
  - Cover page mirrors `LabReportPDF.tsx` exactly.
  - Page header `<Hdr fixed>` repeats on every body page.
  - `pageNumber` `render` callback for automatic page numbering.
  - Document structure (15 pages with the live data set):
    1. Cover
    2. 1 Introduction + 2 Objective (theory eqs 1-2)
    3. 2.1.3 Steady-state amplitude (Eq 3) + 2.1.4 Phase lag (Eq 4) + 2.1.5 Ringdown (Eq 5) + 2.2 Purposes
    4. 3 Method 3.1 Setup
    5. 3.2 Procedure (3 phases)
    6. 4 Raw Data — Tables 1, 2 (parameters + free-osc fit)
    7. Figure 1 (free oscillation)
    8. Table 3 (resonance sweep) + Figure 2 (resonance curves)
    9. Figure 3 (phase lag)
    10. Table 4 (phase runs) + Figure 4 (phase comparison)
    11. 5 Data and Error Analysis 5.1 - 5.5
    12. 5.6 Error Analysis (instrumental + propagation eqs 6-8)
    13. 5.6.5 - 5.6.6 + Table 5 (uncertainty summary)
    14. 6 Conclusion 6.1 (Q&A 1-6)
    15. 6.2 Summary + 7 Appendix (Figure A1, generator footer)

## How the user invokes it

```
1. Open the Exp 4 page in the browser.
2. Configure spring κ, damping γ, driver amplitude A, drive frequency f.
3. Optionally play with Start Driver / Free Oscillation in the live UI.
4. Click "Generate Lab Report" — backend runs the RK4 + matplotlib pipeline.
5. When the green "Exp 4 Lab Report Generated" panel appears, click
   "Download PDF Report" — the React-PDF renders the CUHK-Shenzhen-style PDF.
6. (Optional) "Download All (ZIP)" to also get the Markdown, CSVs, JSON
   summary, and the matplotlib PDF backup.
```
